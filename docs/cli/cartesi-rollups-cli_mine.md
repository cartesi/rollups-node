## cartesi-rollups-cli mine

Mine a new block

```
cartesi-rollups-cli mine [flags]
```

### Examples

```
# Mine a new block:
cartesi-rollups-cli mine
```

### Options

```
      --anvil-endpoint string   address of anvil endpoint to be used to send the mining request (default "http://localhost:8545")
  -h, --help                    help for mine
```
